# AI 工程作战手册

这份手册面向需要快速理解并维护 MGX 工程的 AI/开发同学，结构分为：

1. `overview/` —— 一页纸掌握产品目标与系统架构。
2. `modules/` —— 核心子系统（网关、Agent Runtime、Sandbox、前端）的职责、入口文件与依赖。
3. `playbooks/` —— 高频任务的步骤指引，可直接跟随执行。
4. `instructions/` —— 如何生成/更新本手册的说明。

使用方式：先阅读 `overview/system.md` 打牢全局，再根据当前任务跳转到相应模块或 playbook；若需扩展文档，请遵循 `instructions/maintain.md`。
